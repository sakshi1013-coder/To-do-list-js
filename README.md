# 📝 My Task List

A simple and interactive Task List web app built using **HTML**, **CSS**, and **JavaScript**. It allows users to add and delete tasks dynamically and includes a fun popup welcome message using custom events.

## 🚀 Features

- ✅ Add new tasks
- ✅ Delete individual tasks
- ✅ Custom event-based popup welcome message
- ✅ Styled task list with modern UI
- ✅ Clean code using `addEventListener`, `event delegation`, and custom events
